# HAPI FHIR Server
 [HAPI FHIR](https://hapifhir.io/hapi-fhir/docs/) Server integrated with latest Spring boot application

## Technologies

- Spring Boot 2.3.4.RELEASE
- Maven 3.3.6
- Java 14
- HAPI FHIR 5.1.0
